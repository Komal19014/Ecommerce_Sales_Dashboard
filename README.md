# Ecommerce_Sales_Dashboard

# Project Overview

This project is an interactive Ecommerce Sales Dashboard created using Power BI. The dashboard helps analyze sales performance, profit, quantity sold, and customer behavior across different states, categories, and payment methods.
The goal of this project is to transform raw ecommerce data into clear visual insights that help in better business decision making

# Tools & Technologies Used
1. Power BI – Data visualization and dashboard creation
2. Microsoft Excel / CSV – Dataset
3. Power Query – Data cleaning and transformation
4. DAX – Data calculations and measures

# Dataset
The dataset contains ecommerce sales information including:
-Order details
-Customer names
-Product categories and sub-categories
-Sales amount
-Profit
-Quantity sold
-Payment mode
-State information 

# Project Outcome
-This dashboard helps businesses:
-Track sales performance
-Identify profitable products
-Understand customer purchasing behavior
-Make data-driven decisions
